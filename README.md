# 作業2 前端基礎技術練習

這是個別作業

## 作業說明： 網頁設計 (我的作品及首頁)。

請設計一個標準的單頁式網頁，包含以下區塊：
### html
<header> (導覽列)：放置個人標誌 (Logo) 與導覽選單（首頁、關於我、作品集、聯絡方式）。

<section class="hero"> (主視覺區)：網頁最上方最吸睛的部分。包含一句大標題、簡短自我介紹和一個「我的行動宣言」按鈕，用來介紹自己作品的主軸。

<section class="about"> (關於我)：使用兩欄佈局，一邊放照片，一邊放文字介紹。

<section class="portfolio"> (作品集)：使用卡片式設計 (Card Design) 或分段設計來展示你的練習成果作品。

<footer> (頁腳)：包含版權宣告及社交媒體連結。

### css
將排版設定撰寫在 style.css, 然後在 main.html 中引入
盡可能的豐富你的單頁設計
